# Open OT.

OT_Project pero Libre, para mayor acceso y expansiones.
Para un poco de Contexto: OT_Project fue un proyecto de Tesis de Universidad en el que se basa en la gestión empresarial de los activos de un ambiente, que en este caso, del rubro de Transportes.

La idea de este proyecto actual es dejarlo renovado para dar paso a mejoras y cambios que se vienen a futuro, debido a la industria cambiante.

Claramente, con un nuevo enfoque, podemos eliminar problemas a futuro para una mejor operación a fondo.

La tésis está siendo escrita en Notion, por lo cual, todavía no está listo para publicarla.
